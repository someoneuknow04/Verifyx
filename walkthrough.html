Walkthrough – How to Verify Documents

Follow the steps below to easily verify each document or registration

GST Verification

  1. Visit https://www.gst.gov.in/
  2. Click on “Search Taxpayer” → “Search by GSTIN/UIN”.
  3. Enter the GSTIN number and fill the captcha.
  4. Click “Search” to view legal name, state, status, and registration date.

Udyam / MSME Verification

  1. Go to Udyam Portal.
  2. Click on “Print/Verify” then choose “Verify Udyam Registration Number”.
  3. Enter Udyam Registration Number and mobile number linked.
  4. Submit to view the registration details.

FSSAI (FoSCoS) Verification

  1. Visit https://foscos.fssai.gov.in/
  2. Scroll down to “License/Registration” → “FBO Search”.
  3. Enter License/Registration number, select state & captcha.
  4. Click “Search” to view license details.

IEC (Import Export Code) Verification

  1. Visit DGFT Portal.
  2. Click “Services” → “IEC Profile Management” → “View IEC”.
  3. Enter IEC code and captcha.
  4. Click “Search” to view entity name and branch details.

IRDA Registered Insurance Agents / Brokers

  1. Go to IRDAI Portal.
  2. Click “Intermediary / Agent Search”.
  3. Choose agent type, enter license number or name.
  4. Click “Search” to view registration validity and insurer details.
